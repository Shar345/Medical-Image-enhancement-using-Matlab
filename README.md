# Medical Image Enhancement using MATLAB

## Project Overview

Medical imaging plays an important role in diagnosing and detecting tumors and abnormalities using scans such as **X-rays, CT scans, and MRI**. However, raw medical images often contain noise, low contrast, and unclear edges, which can make it difficult to identify important regions.

This project focuses on **Medical Image Enhancement using MATLAB**. A **Graphical User Interface (GUI)** is developed that allows users to upload medical images, apply enhancement techniques, compare results, and save the final enhanced image.

The main goal of this project is to improve image quality and make tumors and abnormal regions more visible for better analysis.

## Features

- Upload medical images through GUI
- Apply multiple image enhancement techniques
- Compare original and enhanced images
- Improve tumor and abnormality visibility
- Save enhanced output images

## Image Enhancement Techniques Used

### 1. Noise Removal (Denoising)
- Removes unwanted random noise from medical images.
- Preserves important image details and structures.

### 2. Histogram Equalization & CLAHE
- Improves image brightness and contrast.
- Enhances visibility of important regions.
- CLAHE (Contrast Limited Adaptive Histogram Equalization) improves local contrast.

### 3. Image Sharpening
- Enhances edges and fine details.
- Helps highlight tumor regions and abnormalities.

## Working Process

1. User uploads a medical image.
2. The image is processed using enhancement techniques.
3. Noise is reduced using denoising methods.
4. Contrast is improved using Histogram Equalization/CLAHE.
5. Sharpening is applied to highlight important features.
6. Enhanced image is displayed and saved.

## Technologies Used

## Applications

- Medical scan visualization

## Conclusion

This project successfully implements a **Medical Image Enhancement System in MATLAB with an interactive GUI**.

By combining **denoising, histogram equalization, CLAHE, and image sharpening techniques**, the system improves the quality of medical images and makes tumors and abnormalities more visible for diagnosis.

The GUI provides an easy-to-use platform for uploading, processing, comparing, and saving enhanced medical images. This project demonstrates how **digital image processing techniques can assist medical professionals by improving scan clarity and visualization**.
